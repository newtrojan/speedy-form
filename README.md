# AutoGlass Quote System

A monorepo containing the backend and frontend for the AutoGlass Quote System.

## Structure

-   `backend/`: Django REST Framework application (PostgreSQL/PostGIS, Celery, Redis).
-   `frontend/`: React application (Coming Soon).

## Getting Started

See [backend/README.md](backend/README.md) for detailed backend setup and development instructions.

### Quick Setup

```bash
# Start all services
docker-compose up --build
```
